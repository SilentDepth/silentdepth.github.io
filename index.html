工作经历

2013年进入 Teradata 北京公司内部开发支持团队实习，同年正式入职并工作至今。期间参与/主导若干数据相关工具产品的设计开发与维护工作，包括：

某浏览器端 Web 应用开发平台 (2014-2016)

这是一款基于 B/S 架构的 Web 应用开发工具，同时也是此类应用的底层支持环境。该平台主要面向 SQL 开发人员，以可视化、拖拽式操作完成应用界面和业务逻辑的开发，并集成调试与部署的工作流程，使非专业 Web 应用开发人员也可以完整快速地实现用户交互、SQL 外逻辑处理、数据库操作等程序功能。我先后负责其1.0版本的测试与维护、1.1版本的开发与项目支持、1.2版本的新功能设计与开发工作。

某银行数据管理平台IV/V/VI/VII期开发实施 (2013-2016)

某银行的一系列数据管控方案的实施，使用上述平台进行开发和部署。我在其中负责的工作主要体现在以下两个方面：

若干业务模块的设计和实施——在对业务需要的理解之上，利用上述平台设计并实现业务操作流程，响应业务需要，帮助客户构建数据管控体系，提高数据管控质量。

相关底层技术开发和支持——遇到开发平台无法或不能良好地实现业务需求时，编写必要的前后台程序，同时在其他项目成员遇到问题时提供技术支持和错误修复，以保证项目工作的正常进行。这一方面也包含了对上述开发平台的维护更新工作，项目上证实的功能代码将被最终合并入开发平台的代码中。

除此之外我还负责项目的质量控制、项目自动化实现、客户技术对接等工作。

基于百度地图的热图演示程序 (2014)

为了让客户了解 Aster（Teradata 的一个面向数据分析的数据库产品）的优势和价值，以银行选址辅助分析为主题启动的一个概念验证项目。我独自完成了演示程序（Web 应用）的设计和开发，以热图的形式基于百度地图实现了分析数据的可视化展示，并参加了某银行的大数据成果展览。

某 ETL 调度工具 (2013)

实习时参与的项目，主要负责部分功能的业务逻辑实现和 SQL 编写，以及界面设计和测试工作。
